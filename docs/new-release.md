# Updating versioning for package managers and whatnot

- App metainfo: `extra/linux/dev.phidi.phidi.metainfo.xml`
- macOS plist (`CFBundleShortVersionString`): `extra/macos/Phidi.app/Contents/Info.plist`
- Rust: `Cargo.toml`
- Obviously changelog: `CHANGELOG.md`
- RPM spec: `phidi.spec`
- Windows wix spec (`<Product [...] Version=X.X.X>`): `extra/windows/wix/phidi.wxs`
